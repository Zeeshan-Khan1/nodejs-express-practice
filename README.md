# nodejs-express-practice

A collection of hands-on practice projects for learning **Node.js** and **Express.js** from basic to advanced.  
This repository includes essential concepts, useful commands, and mini-projects including API development and MongoDB integration.

---

## Folder Structure

- **01-nodejs-express-introduction**  
  Basics of Node.js, Express, routing, and server setup.

- **02-essential-packages-commands**  
  Important NPM packages, installation commands, and usage.

- **03-API-&-MongoDB**  
  Simple API project with CRUD operations and MongoDB (Mongoose).

- **Will be add**
---

## Purpose

This repository is created to practice backend development and improve understanding of:

- Node.js fundamentals  
- Express routing and middleware  
- REST API development  
- MongoDB & Mongoose  
- NPM package management  

---

## Technologies Used

- Node.js  
- Express.js  
- MongoDB / Mongoose  
- Nodemon
