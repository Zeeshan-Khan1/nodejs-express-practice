# Install Express
### What it does:

Adds the Express framework to create routes, APIs, middleware, server, etc.

###Syntax:
```js
npm install express
```
## Steps:

### Run in terminal: npm install express

### Adds Express to your project

### Adds dependency inside package.json

Result added in package.json:
"dependencies": {
  "express": "^4.x.x"
}
